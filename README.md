# proyecto-DSM


## Como clonar el proyecto.

```jsx
git clone URL
```

## Comandos importantes.

```jsx
git fetch --> Sirve para actualizar el repositorio local con respecto al remoto

git pull --> Sirve para bajar los cambios del repositorio remoto al local

git add -A --> Sirve para agregar los nuevos cambios al repositorio local

git commit -m "comentario" --> Sirve para agregar un commit o comentario al cambio realizado

git push --> Sirve para subir los cambios de local a repositorio remoto (Importante: Verificar siempre la rama donde hace push)

git branch --> Muestra una lista de las ramas del proyecto

git checkout nombreRama --> Sirve para cambiarnos de rama
```
